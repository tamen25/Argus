# SPA-004 — root spans are not CLIENT spans

**Impact:** Important (weight 30) · **Target:** span · **Source:** spec ·
**Confidence:** sampled only

A root span with kind CLIENT means the downstream server never continued the
trace — propagation broke at the first hop.

**Detection.** Pure per-item check: violation when `!span.has_parent &&
span.kind == 'client'`. Default threshold_ratio 0 (any occurrence fails the
service); raise it if head sampling frequently drops server sides.

**Remediation.** Template `broken-context-propagation`.
