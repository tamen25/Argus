# MET-002 — metrics have a unit

**Impact:** Important (weight 30) · **Target:** metric · **Source:** spec ·
**Confidence:** sampled only

Metrics MUST carry a unit of measurement. `1` (dimensionless) is valid UCUM;
empty is not.

**Fidelity note.** Full UCUM grammar validation is out of scope for v0.1 —
this checks non-emptiness only (DECISIONS.md). Pair with
[MET-005](met-005.md), which catches units embedded in names instead.

**Remediation.** Template `missing-metric-unit`: set the unit in the
instrument constructor (`WithUnit("s")` etc.).
