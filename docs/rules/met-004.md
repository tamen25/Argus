# MET-004 — histogram buckets are consistent per metric

**Impact:** Normal (weight 20) · **Target:** metric · **Source:** spec ·
**Confidence:** sampled only

All histogram series of one metric name MUST share a bucket layout — mixed
layouts break cross-instance quantile aggregation.

**Detection.** Each histogram point's bucket layout is canonicalized to a
signature string; an HLL sketch counts distinct signatures per
(service, metric) (`histogram_bucket_signatures` aggregate). Cardinality > 1
fails. The spec's 14-day horizon is approximated by the engine's tumbling
window, same as [LOG-001](log-001.md).

**Remediation.** Template `histogram-bucket-mismatch`: align bucket config in
one place (collector view or shared SDK config), version it.
