# Argus

**CI for reliability.** Argus scores your telemetry, prices it, backtests your alerts against
history, and proves — with an AI-agent benchmark — whether your observability can actually
diagnose an incident.

Argus is an independent open-source implementation of the
[Instrumentation Score specification](https://github.com/instrumentation-score/spec), extended
with cost attribution, alert backtesting, remediation generation, and a live-environment agent
benchmark (scenario-compatible with ITBench).

Status: **Phase 0 — Foundation**. See the [master build plan](argus-master-build-plan.md) for the
full specification and roadmap, and the [quickstart](quickstart.md) for the dev environment.
